# BIKAS88 - Premium Gaming & Wallet Management Platform

একটি আধুনিক এবং মডুলার গেমিং ও ওয়ালেট ম্যানেজমেন্ট ব্যাকএন্ড সিস্টেম, যা এক্সপ্রেস জেএস (Express.js) এবং ফায়ারবেস ফায়ারস্টোর (Firebase Firestore) দিয়ে তৈরি।

## ফোল্ডার স্ট্রাকচার
- `config/` - ডাটাবেজ এবং ক্লাউড কনফিগারেশন।
- `middleware/` - সিকিউরিটি এবং রোল ভ্যালিডেশন লেয়ার।
- `models/` - ডাটাবেজ স্কিমা মডেল।
- `routes/` - এপিআই এন্ডপয়েন্টস।

## কীভাবে রান করবেন?
১. ডিপেন্ডেন্সি ইনস্টল করুন: `npm install`
২. `.env` ফাইল তৈরি করে ক্রেডেনশিয়াল সেট করুন।
৩. সার্ভার চালু করুন: `npm start`
